About us

Our application was created in June 2021.
It is a web application that keeps track of drugs in a pharmacy.
The application allows you to track information about medicines available in a pharmacy. The computer system includes a user interface, a database server that will store information about drugs, the application server.
Depending on the type of authentication (administrator or user), the user will be allowed to manage the information or just view, upload or download it. The system will provide the ability to upload or download medication information and manage it.
